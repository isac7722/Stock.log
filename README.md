### 매매일지 기록 서비스
### 개발기간 : 2024.01.20 ~ 
<br>
- 기록해서 돈 버는 습관, <b>Stock.Log</b>는 초보 주식 투자자가 매매일지를 작성하는데 도움을 주는 서비스입니다.
- 주식 종목의 매수와 매도에 따른 정보를 입력해 발생하는 총자산 변화 등을 이용자에게 제공합니다.
